Quick Start

Get started with this simple example:

from UsableFunctions import UsableFunctions as u

# Basic calculator example
result = u.calculator(5, 7, '+')
print(result)  # Output: 12

# Generate a random password
password = u.get_random_password(length=12)
print(password)

# Convert Python script to executable
u.py_to_exe('your_script.py')

Available Functions

  • calculator(a, b, operator)

    Perform basic arithmetic operations

  • quest(player, difficulty)

    Interactive math quiz game with difficulty levels 1-3

  • py_to_exe(file)

    Convert Python scripts to standalone executables

  • get_random_password(length)

    Generate secure random passwords

  • factorial(n)

    Calculate factorial of a number

  • is_pressed(key)

    Check if a specific key is pressed

  • install_package_git(url, creator, repo)

    Install Python packages from Git repositories

  • write_in_new_file(filename, content)

    Create and write content to new files

  • read_from_file(filename)

    Read content from existing files

  • delete_file(filename)

    Delete a file from the system

  • rename_file(old_name, new_name)

    Rename an existing file

  • move_file(source, destination)

    Move a file to a new location

  • Circle

    Class for circular motion calculations (velocity, period, frequency)

  • get_ip_address()

    Get the IP address of the current machine

  • get_mac_address()

    Get MAC address of the current machine

  • get_machine_info()

    Get information about the current machine(hostname, ip address, mac address, platform, python version)

  • get_memory_info()

    Get memory usage information

  • optimize_system()

    Optimize system performance by clearing memory cache. WARNING: Use with caution! This may affect system stability and you need to reload the internet

  • get_disk_info()

    Get disk usage information

  • get_cpu_info()

    Get CPU usage information

  • system_panel()

    System information panel

  • Class FileConverter

    In this class, you can convert files from one format to another(png -> jpg, jpg -> png, etc.)

  • FileConverter.gui

    Class with gui functions, like converter(mode) or gui() - universal GUI launcher
