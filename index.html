Installation

From ZIP

Download ZIP 
pip install -r requirements.txt

As a Package (Recommended)

pip install --upgrade git+https://github.com/Artemon0/UsableFunctions.git

Quick Start

Get started with this simple example:

from UsableFunctions.UF import UsableFunctions as u

# Basic calculator example
result = u.calculator(5, 7, '+')
print(result)  # Output: 12

# Generate a random password
password = u.get_random_password(length=12)
print(password)

# Convert Python script to executable
u.py_to_exe('your_script.py')

Available Functions

  • calculator(a, b, operator)

    Perform basic arithmetic operations

  • quest(player, difficulty)

    Interactive math quiz game with difficulty levels 1-3

  • py_to_exe(file)

    Convert Python scripts to standalone executables

  • get_random_password(length)

    Generate secure random passwords

  • factorial(n)

    Calculate factorial of a number

  • is_pressed(key)

    Check if a specific key is pressed

  • install_package_git(url, creator, repo)

    Install Python packages from Git repositories

  • write_in_new_file(filename, content)

    Create and write content to new files

  • read_from_file(filename)

    Read content from existing files
