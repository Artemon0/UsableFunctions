Installation

Choose your preferred installation method:

From ZIP

Download ZIP 
pip install -r requirements.txt

As a Package (Recommended)

pip install --upgrade git+https://github.com/Artemon0/UsableFunctions.git

Quick Start

Get started with this simple example:

from UsableFunctions.UF import UsableFunctions as u

# Basic calculator example
result = u.calculator(5, 7, '+')
print(result)  # Output: 12

# Generate a random password
password = u.get_random_password(length=12)
print(password)

# Convert Python script to executable
u.py_to_exe('your_script.py')

Available Functions

  • calculator(a, b, operator)

    Perform basic arithmetic operations

  • quest(player, difficulty)

    Interactive math quiz game with difficulty levels 1-3

  • py_to_exe(file)

    Convert Python scripts to standalone executables

  • get_random_password(length)

    Generate secure random passwords

  • factorial(n)

    Calculate factorial of a number

  • is_pressed(key)

    Check if a specific key is pressed

  • install_package_git(url, creator, repo)

    Install Python packages from Git repositories

  • write_in_new_file(filename, content)

    Create and write content to new files

  • read_from_file(filename)

    Read content from existing files

  • delete_file(filename)

    Delete a file from the system

  • rename_file(old_name, new_name)

    Rename an existing file

  • move_file(source, destination)

    Move a file to a new location

  • math_factorial(n)

    Calculate factorial using math module

  • while_calculator()

    Interactive calculator with continuous input

  • get_progress_bar(iterable)

    Create a progress bar for iterations

  • update_this_program()

    Update the UsableFunctions package

  • create_new_file(filename, content, filepath)

    Create a new file with optional content

  • read_file_content(filepath)

    Read the entire content of a file

  • install_package(package)

    Install a Python package using pip

  • uninstall_package(package)

    Uninstall a Python package using pip

  • run_py_file(filepath)

    Execute a Python file

  • run_py_file_with_args(filepath, args)

    Execute a Python file with command line arguments

  • run_python_code(code)

    Execute Python code as a string

  • check_password(username, password, correct_username, correct_password)

    Validate username and password credentials

  • Circle

    Class for circular motion calculations (velocity, period, frequency)
